 # Song2Quiz

Song2Quiz is an AI-powered tool that transforms songs into educational quizzes. It analyzes song lyrics and musical elements to generate engaging quiz questions that test knowledge of:

- Lyrics and meaning
- Musical structure and composition
- Cultural and historical context
- Literary devices and themes
